#Descrições do projeto de interfaces

*Empresa: *Clínica veterinária. Foi levantado que a empresa possui um sistema chamado SisVet. 
Este sistema controla somente fluxo de caixa, sendo que qualquer usuário pode alterar os valores de entrada e saída.
Hoje é realizado o atendimento médico veterinário, onde todo o histórico do paciente (animais) é armazenado em fichas de papel,
 após o atendimento, é realizado uma ficha de cadastro do animal com prontuário, receituário e com dados do cliente.

*Os produtose medicamentos não são controlados por um sistema, muitas vezes é receitado medicamentos que não contém mais no estoque, 
ocasionando em perda de vendas e também no desconforto de o cliente não poder comprar o medicamento no ato. 
Com esta falta de controle específico, a clínica está tendo prejuízos.
