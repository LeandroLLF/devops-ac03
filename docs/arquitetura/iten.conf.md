# codigo fonte 
* no codigo fonte será usado a linguagem sql server para criar um banco de dados para o nosso cliente, com o banco de dados
o cliente pode armazenar todo tipo de informaçao de sua empresa 

* o sistema sera usado da seguinte maneira, será criada tabelas relacionadas a empresa, por exemplo, estamos com uma empresa de 
pet shop seria criado uma tabela chamada cachorro onde armazena todos dados do cachorro e uma tabela chamado médico onde armazena todos dados 
do médico , quando o funcionário puxar a tabela cachorro automaticamente sera puxado a tabela médico então mostra quantos cachorros o médico 
atendeu ou mostra qual médico atendeu esse cachorro .
